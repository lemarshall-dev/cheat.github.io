Position Rule | CSS

Rule Action
position This element will allow you to change the top, right, bottom and left postioning of selected element.
top: 20px This will add 20px of space from the top. Moving the selected element down 20px
right: 20px This will add 20px of space from the right. Moving the selected element left 20px
bottom: 20px This will add 20px of space from the bottom. Moving the selected element up 20px
left: 20px This will add 20px of space from the left. Moving the selected element right 20px

Link States | CSS

The ordering of link state pseudo-class rules is important to reveal the proper information. When a user hovers and then clicks a link, those styles should always override the static styling surrounding a user’s history with the link (unvisited :link and :visited). The proper order of these rules is:

  • :linked
  • :visited
  • :hover
  • :active

This ordering will ensure that the rules cascade properly and the user can receive the most applicable visual feedback about the state of the link.
